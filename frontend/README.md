
# Book Shop

Интернет-магазин книг Book Shop - это сервис с удобным и современным адаптивным интерфейсом для просмотра каталога книг, управления его позициями с возможностью добавлять, изменять и удалять записи в системе.

## Разработка

При первом запуске проекта в директории `js-frontend-middle-test-assignment` необходимо последовательно запустить следующий набор команд:

```bash
cd frontend
npm install
npm run dev
```

Для сборки проекта необходимо запустить следующую команду:

```bash
npm run build
```

Для развертывания окружения разработки используйте команду `docker compose up` в директории проекта:

```bash
sudo docker compose up
```

Также нужно создать переменные окружения для успешной работы проекта (см. раздел Переменные окружения).

После развертывания окружения разработки клиентская часть проекта будет доступна по адресу
http://localhost:8080/.

Дополненная документация OpenAPI по используемым API endpoints в проекте находится в файле `openapi.yaml` директории `docs`.

## 🔧 Переменные окружения

Создайте файл `.env` в каталоге frontend с данным содержимым:

```env
VITE_API_URL=http://localhost:8080  # Базовый URL API
VITE_API_PATH=api                   # Префикс API
```
Это необходимо для корректной работы API и авторизации пользователя в системе.
